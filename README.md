# Superstore Sales Analysis – End-to-End Capstone

This project analyzes the Superstore retail dataset and generates visualizations to explore sales trends, category performance, and monthly patterns. It automatically produces charts for key metrics and insights.

---

## Features
- Clean and preprocess Superstore dataset
- Generate Monthly Sales Trend chart
- Create Sales by Category bar chart
- Extract actionable business insights
- Fully reproducible analysis workflow
- Easy to extend with additional charts or KPIs

---

## How to Run

### Install dependencies:
```bash
pip install pandas numpy matplotlib seaborn
```

###Project Structure
```
```bash
superstore-capstone/
├── superstore_capstone.ipynb     
├── README.md                     
├── data/
│   └── superstore.csv             
├── outputs/
│   ├── monthly_sales.png         
│   └── category_sales.png
```
###Outputs

-'monthly_sales.png' — Line chart showing sales trend by month

-'category_sales.png' — Bar chart showing sales by category


###Actionable Insights

-Technology category generates the highest revenue → prioritize inventory and promotions

-Sales show clear monthly patterns → plan seasonal marketing campaigns

-Some categories have high sales but low profit → review pricing and costs
